# SolCipher Kiro

Privacy-first dApp for encrypted file sharing on Hedera + IPFS with Kiro (KILT) DIDs and revocable VCs.

## Features
- Client-side AES-256 encryption
- IPFS storage
- Kiro-based access control with revocation
- Hedera HCS logging (optional)

## Setup
1. `npm install`
2. Copy `.env.example` to `.env` and fill in values
3. `npm run dev`

## Deployment
- Contracts: `npx hardhat deploy --network hedera_testnet`
- Frontend: Deploy to Vercel
